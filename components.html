Button Library

Alle Button-Styles zum Anschauen und kopieren. Jede Variante steht im selben Stylesheet (src/styles.css) und nutzt das gleiche Tracking-Attribut data-link="...". Klicks werden automatisch erfasst, wenn Supabase konfiguriert ist.

1. Basic Button

Der Standard. Eine Zeile, zentrierter Text, weisser Hintergrund.

Mein Link 
<a href="https://..."
   class="link-button"
   data-link="Name fuers Tracking"
   target="_blank" rel="noopener">
    <span class="link-button__text">Button-Text</span>
</a>

2. Stacked Button

Titel + Untertitel auf zwei Zeilen. Gut fuer Newsletter, Substack, Communities.

Newsletter Woechentliche Insights und Updates 
<a href="https://..."
   class="link-button link-button--stacked"
   data-link="Name"
   target="_blank" rel="noopener">
    <span class="link-button__text">Titel</span>
    <span class="link-button__subtitle">Untertitel</span>
</a>

3. Outline Button

Transparent mit weissem Rand. Setzt Akzente, wenn du mehrere weisse Buttons hast.

Sekundaerer Link 
<a href="https://..."
   class="link-button link-button--outline"
   data-link="Name"
   target="_blank" rel="noopener">
    <span class="link-button__text">Button-Text</span>
</a>

Funktioniert am besten als 2. oder 3. Option neben einer Highlight-Card.

4. Highlight Card

Foto links, Eyebrow + Titel + Subtext + CTA rechts. Fuer dein wichtigstes Angebot — Beratung buchen, Hauptprodukt kaufen, Kurs starten.

1:1 mit dir
Persoenliche Beratung
Ein Satz, der den Wert erklaert
Termin sichern →
 
<a href="https://..."
   class="consulting-card"
   data-link="Name"
   target="_blank" rel="noopener">
    <div class="consulting-card__img" role="img" aria-label="..."></div>
    <div class="consulting-card__content">
        <div class="consulting-card__eyebrow">EYEBROW</div>
        <div class="consulting-card__title">Hauptangebot</div>
        <div class="consulting-card__sub">Beschreibung</div>
        <div class="consulting-card__cta">CTA-Text &rarr;</div>
    </div>
</a>

Bild aendern: in src/styles.css bei .consulting-card__img das background-image auf deine eigene Datei umstellen (z.B. url('../assets/dein-bild.webp')).

5. Video Card (YouTube Auto-Update)

Zeigt automatisch dein neuestes Longform-Video (Default: ueber 5 Min, Shorts werden gefiltert). Thumbnail + Titel werden serverseitig via YouTube API geholt und am CDN gecached, damit du im API-Quota bleibst.

Neuestes Video Wird automatisch via YouTube API aktualisiert 
<a id="latest-video-link"
   href="https://www.youtube.com/@deinkanal"
   class="video-card"
   data-link="Neuestes Video"
   target="_blank" rel="noopener">
    <img id="latest-video-thumb" class="video-card__thumb" src="" alt="Thumbnail">
    <div class="video-card__overlay">
        <svg class="video-card__play" ...>...</svg>
    </div>
    <div class="video-card__info">
        <span class="video-card__label">Neuestes Video</span>
        <span class="video-card__title" id="latest-video-title">Wird geladen...</span>
    </div>
</a>

Setup-Guide: docs/02-setup-youtube.md — du brauchst einen YouTube Data API v3 Key, die Channel-ID und musst die ID in netlify/functions/latest-video.js mappen.

6. Product Card

Bild links, Text rechts. Fuer Empfehlungen, Affiliate-Links, Buecher, Tools.

Produkt-Name 
<a href="https://shop-link.com"
   class="product-card"
   data-link="Produktname"
   target="_blank" rel="noopener">
    <img class="product-card__image" src="assets/produkt.jpg" alt="Produkt">
    <span class="product-card__text">Produkt-Name</span>
</a>

Produkt-Sektion ist standardmaessig versteckt. Aktivieren: in src/styles.css die Zeile .products { display: none; } aendern zu display: block;.

Tracking — Wie das funktioniert

Jedes Element mit data-link="..." wird automatisch beim Klick getrackt. Der Wert landet im Dashboard als "Most Clicked Links". 

// Page-View beim Laden
{ event_type: "view", channel: "de", country: "DE", device: "mobile",
  referrer: "instagram.com", utm_source: "instagram" }

// Klick auf einen Link mit data-link="Newsletter"
{ event_type: "click", link_name: "Newsletter", channel: "de", ... }

Details: docs/03-setup-tracking.md. SQL fuer die Tabelle: docs/01-setup-supabase.md.
